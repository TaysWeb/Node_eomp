Home Page Route Links

Users
Method/s Path Description
GET /Users Gets all the users from the Users Table
POST /Users/:id Gets an individual User
DELETE /UserDelete/:id Delete's a user using the userId
PUT /UserUpdate:id Updates a user in the Users table using their userId
